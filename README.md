## Описание
Трекер задач.

## Запуск
Приложение проверено на базе Python 13.
1) Заполните .env файл по примеру .env.example.
2) Запустите docker-контейнер: `docker-compose up --build`. Контейнер запустит PostgreSQL и само приложение.

## Технологии
- Python 3.11 + FastApi + sqlalchemy
- React
- PostgreSQL
- Docker для контейнеризации

## Функции
1) Регистрация, авторизация.
2) Создание команды, добавление в нее участников.
3) Создание/редактирование/удаление проектов.
4) Создание стримов внутри проектов.
5) Создание внутри стримов задач и целей.
6) Страница с просмотром задач проекта и задач, назначенных именно вам.
